# 快速开始

WARNING

要求 Node >= 8.0.0。

安装

npm i --save ts-lib-basic
# OR
yarn add ts-lib-basic

使用

app.ts:

import * as express from 'express'

const app = express()
app.use(middleware)

app.get('/example', (req, res) => {
  res.send(res.locals)
})

ex.listen(3000, () => {
  console.log('Example app listening on port 3000!')
})

// 访问 localhost:3000/example 即可
